A Python-based tool that connects to the Spotify API to generate custom playlists by genre. It pulls track data and builds personalized recommendations based on user input and listening preferences.

A console-based poker game built in Java, complete with logic for gameplay, betting rounds, and randomized card dealing. Focused on simulating strategy and game flow.

A searchable database of ballets with detailed information and a homepage design. Users can search by title, choreographer, or genre. Built with a clean interface and functional backend.

An interactive learning site explaining the exposure triangle — ISO, aperture, and shutter speed. Includes photo examples, a quiz, and a custom-built camera simulator. The site is narrated by “Stacey,” a fictional influencer hiring users to take her photo, adding a humorous narrative layer.
